# Canvas tracking video cam

## Setup

- Install requirements

```
python -m venv venv
. ./venv/bin/activate

pip install -r requirements.txt
```

- Prepare AR markers

```
python ar-canvas-tracker.py
```

AR markers are generated in markers directory.


## Run

```
python ar-canvas-tracker.py
```
